## 这个项目是做什么的？

互联网上「免费」二字经常被滥用：有的限时、有的要绑卡、有的偷偷降级。  
**Whats Free** 做三件事：

1. **分类整理** — 按场景（开发、设计、学习、影音、云资源等）列出值得收藏的真免费项。  
2. **写明条件** — 是否需要信用卡、是否教育认证、额度/期限大概怎样，避免浪费你的时间。  
3. **持续维护** — 欢迎 PR 与 Issue：新增、纠错、标注「已变脸」的服务。

这不是爬虫商城，也不卖课；只是一份 **社区共建清单**，方便你我他快速检索。
